centos安装MySQL（外部导入rpm包）

2025-10-26

安装步骤

首先去MySQL yum源官网查看rpm源的文件名，然后下载Yum源

wget https://repo.mysql.com/mysql84-community-release-el10-2.noarch.rpm 
# mysql84-community-release-el10-2.noarch.rpm替换为官方的文件名

然后执行

sudo rpm -ivh $filename # $filename替换为文件名

接着执行

sudo yum install mysql  # 安装服务端
sudo yum install mysql-community-server # 安装MySQL服务端
systemctl start mysqld  # 启动mysqld服务
systemctl enable mysqld # 开机自启

创建账户

账户密码肯定要修改，首先执行

sudo cat /var/log/mysqld.log

找到root@localhost字样的地方会出现root@localhost: password，接着使用

mysql -uroot -p # 简单登录

后输入刚刚看到的密码即可登录，登录后使用下面的代码执行密码重置

ALTER USER 'root'@'localhost' IDENTIFIED BY 'PASSWORD' -- 密码要包括大小写，符号，数字

当输出Query OK即修改完成，接着输入quit或者exit退出MySQL当前状态，再重新使用下面的指令重新进入MySQL。

mysql -h localhost -u root -P 3306 -p   
# 参数解释
# -h mysql连接地址
# -u 用户名
# -P （异于-p）MySQL的端口号（默认3306）
# -p （异于-P）使用密码登录MySQL

输入密码登录MySQL即可使用。
